Experiment 3: Interactive Admin Dashboard

• Aim  
  To design and develop an interactive Admin Dashboard using HTML, CSS and JavaScript with theme control and navigation.

• Objectives  
  - To create a dashboard layout using CSS grid  
  - To design a sidebar navigation system  
  - To implement page switching using JavaScript  
  - To implement light mode and dark mode toggle  
  - To store theme preference using localStorage  
  - To make the dashboard responsive for mobile screens  

• Tools / Technologies Used  
  - HTML5  
  - CSS3  
  - JavaScript  
  - Visual Studio Code  
  - Web Browser (Chrome / Edge)  

• Description  
  This experiment demonstrates an interactive Admin Dashboard interface that includes navigation between different sections such as Dashboard, Reports, Users and Settings. It also provides a theme switch option between Dark Mode and Light Mode. The dashboard is responsive and works properly on both desktop and mobile screens.

• Files Included  
  - index.html  
  - style.css  
  - script.js  

• Output  
  A responsive interactive dashboard with navigation and theme switching.
