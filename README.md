## ✨ Animations Included

- 🌊 **Keyframe Animations** — Multi-step, looping visual effects
- 🔄 **CSS Transforms** — rotate, scale, translate, skew
- 💎 **3D Transforms** — perspective-based card flips and depth
- ⚡ **Transition Effects** — Smooth state changes on interaction
- 🌈 **Gradient Animations** — Shifting color backgrounds
- 🎪 **Hover Effects** — Micro-interactions on user hover

## 📖 What You'll Learn

```css
/* Example: Keyframe animation */
@keyframes pulse {
  0%, 100% { transform: scale(1); opacity: 1; }
  50%       { transform: scale(1.05); opacity: 0.8; }
}

.element { animation: pulse 2s ease-in-out infinite; }
```
